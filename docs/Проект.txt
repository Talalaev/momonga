--------------------------------------------
------------ Логика всей работы ------------
--------------------------------------------

На любом экране можно изменить группу в которой мы находимся.

Экраны:
1. Первый экран - ШНЯГИ
Выбираем шнягу -> Две кнопки Покупки/Продажи

2. Второй экран - КАТЕГОРИИ
Выбираем категорию -> Много кнопок.

3. Третий экран - ЦЕНА
Вводим цену -> инпут для ввода. 
Справа от инпута отображается значек текущей валюты.
Если нужно изменить валюту клацаем на значек выбираем из списка избраных валют.

4. Четвертый экран - ДАТА
Подтверждаем дату по-умолчанию либо устанавливаем свою дату.

5. Пятый экран - ПОДТВЕРЖДАЕМ
Жмем на кнопку сохранить.

--------------------------------------------
------------ Список коллекций --------------
--------------------------------------------

1. Пользователи
У пользователя есть валюта по умолчанию, а так же список избранных валют для быстрого переключения.
Дата покупки по умолчанию проставляется сервером, если пользователь явно не укажет дату.

2. Шняги
3. Группы
Группы видны всем т.е. любой пользователь может состоять в любой группе

4. Категории
Есть набор дефолтных категорий они хранятся отдельно от пользователя. 
Так же есть возможность создавать свои категории для пользователя.

5. Валюты
6. Иконки

--------------------------------------------
------------ Список документов -------------
--------------------------------------------

1. Пользователь
- имя: строка
- почта: строка
- пароль: строка
- группы: список групп в которых состоит пользователь
- категории: список созданых пользователем категорий
- валюта: валюта в которой пользователь ведет учет по-умолчанию
- избранная валюта: список валют для быстрого переключения. максимально 4 валюты.
- уведомления: список уведомлений
- права: номер

2. Шняга
- имя: покупка|продажа
- категория: идентификатор
- цена: валюта
- дата: строка UTC ISO-формат
- группа: идентификатор

3. Группа
- имя: строка
- идентификатор: номер
- создатель: идентификатор
- админы: список идентификаторов

4. Категория
- имя: строка
- идентификатор: номер
- иконка: код

5. Валюта
- имя: строка
- обозначение: код
- идентификатор: номер

6. Иконка
- имя: строка
- идентификатор: номер
- код: строка
